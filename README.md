# LMS Frontend (React + Vite)

A modern frontend for our Learning Management System that organizes and enhances YouTube-based learning with features like to-do tracking, progress monitoring, AI summaries, quizzes, and note-taking.

> ⚡ Built with **React + Vite**  
> 📚 Backend: Laravel (API-driven)  
> 🧠 AI Integration: Gemini API

---

## 🚀 Features

- 📺 Embed and organize YouTube videos/playlists
- ✅ Track learning with a to-do list and progress bar
- 🧠 AI-powered summaries and auto-generated quizzes
- 📝 Write personal notes tied to each video
- 🔍 Search and save YouTube content within the app
- 📅 Goal-setting and timeline tracking

---

## 🧱 Tech Stack

| Tool        | Purpose                       |
|-------------|-------------------------------|
| React       | UI framework                  |
| Vite        | Fast build tool               |
| Tailwind CSS (optional) | Styling            |
| Axios       | API calls                     |
| React Router| Routing (if used)             |
| Gemini API  | Summarization & quiz engine   |
| YouTube Data API | Fetching video content   |
